# 0x0B. C - malloc, free

## Description
This project covers dynamic memory allocation in C using `malloc` and `free`. It explores how to allocate memory on the heap at runtime, manage memory layouts, and prevent memory leaks.

## Technologies
* Language: C (C99 standard)
* OS: Ubuntu 20.04 LTS
* Compiler: gcc
* Style Guide: [Betty Style](https://github.com)

## Files & Tasks

| File | Description |
| --- | --- |
| `main.h` | Header file containing all function prototypes. |
| `0-create_array.c` | Creates an array of chars and initializes it with a specific char. |
| `0-main.c` | Test file for the `create_array` function. |

## Compilation
Compile the code using the following flags:
```bash
gcc -Wall -pedantic -Werror -Wextra -std=gnu89 0-main.c 0-create_array.c -o a
```
